## Unckless 2015 results detail (18 elastic net LOO-CV runs)

All runs use `scripts/run_dgrpool_phenotype.py --sex M` (`--spectral-sex` at its default, `F`). "Collapsed" means the script's own diagnostic fired: predicted values had standard deviation below 20% of the true values' standard deviation, i.e. the model gave up and predicted close to the training mean rather than finding any real signal.
